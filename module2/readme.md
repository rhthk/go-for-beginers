# Module 2: Go Core Topics - Practice Questions

This file contains 20 programming questions covering core Go topics: `var / :=`, `const`, `int / float64 / string / bool`, arrays, slices, maps, `if`, `switch`, `for`, functions, multiple returns, `defer`.

---

### 1. Variable Swap

**Description:** Write a program to swap two integers using variables.  
**Example:**  
Input: `a=5, b=10`  
Output: `a=10, b=5`

---

### 2. Temperature Converter

**Description:** Read a float64 temperature in Celsius and convert it to Fahrenheit using a constant for the conversion factor.  
**Example:**  
Input: `25`  
Output: `77`

---

### 3. Sum of Array

**Description:** Given an array of integers, calculate the sum of all elements.  
**Example:**  
Input: `[1, 2, 3, 4]`  
Output: `10`

---

### 4. Max in Slice

**Description:** Find the maximum value in a slice of float64 numbers.  
**Example:**  
Input: `[1.5, 2.7, 0.3]`  
Output: `2.7`

---

### 5. Count True

**Description:** Count the number of `true` values in a slice of bool.  
**Example:**  
Input: `[true, false, true]`  
Output: `2`
